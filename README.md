
## Fix: Grub Boot Menu

Fix: Windows not showing in grub boot menu | kali Linux


### Installation

Download grub by cloning the Git repository :
```bash
$ git clone git@github.com:aonetecha/Fix-Grub-Boot-Menu.git
```
    
## Usage
```bash
$ cd Fix-Grub-Boot-Menu
$ sudo chmod +x grub.sh
$ sudo ./grub.sh
